Printing on a TSC label printer
  1. Install TSC's Windows driver (Seagull Driver Wizard, from TSC's website) for your printer model.
  2. In the printer's driver properties, create/select a label stock sized 152 × 101 mm (matches this design).
  3. Click Print Label above, choose the TSC printer in the print dialog, set margins to "None" / paper size to the 152×101 label stock, and print.
  4. For silent/automatic printing without the dialog box (e.g. from a barcode scanner trigger), TSC printers can also be driven with raw TSPL commands via a small local print-agent — ask if you'd like this added as a next step.
  5. For a batch, use Print All below — it sends every queued label as one print job. In the print dialog, make sure "Pages per sheet" is set to 1 so each label prints on its own sheet.
